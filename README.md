# 💸 ExpenseBot — AI Expense Tracker PWA

A Claude-powered expense tracker built as a Progressive Web App.

## Setup
1. Visit your deployed Vercel link
2. Enter your Claude API key (get one free at console.anthropic.com)
3. Start tracking: "Spent ₹500 on food"

## Storage
All data saves to your phone's localStorage — persists across sessions.
